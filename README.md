# spicedb-schema-parser

`spicedb-schema-parser` is a TypeScript library designed to parse SpiceDB schema files into an Abstract Syntax Tree (AST). This library provides a robust API for developers and tools to inspect and interpret SpiceDB schemas with ease.

## Features

- **Parse SpiceDB Schema Text**: Convert schema text into a structured AST.
- **Inspect and Interpret Schemas**: Easily navigate and understand the structure of SpiceDB schemas.
- **Integration-Friendly**: Designed to be used as a foundational tool for other applications that interact with SpiceDB schemas.

## Installation

To install `spicedb-schema-parser`, use npm:

```bash
npm install spicedb-schema-parser
```
